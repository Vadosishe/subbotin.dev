# Hello, World!

This is my first test post on the new site. I decided to make a minimalist design in the **Bento Grid** style, using modern technologies.

## Why Next.js?

In 2026, `Next.js` remains the best choice for creating fast and optimized React applications. App Router significantly simplifies routing and layout management.

### Tech Stack:

*   **Next.js (App Router)** - The power of React Server Components.
*   **Tailwind CSS v4** - Maximum convenience working with styles directly in markup.
*   **Framer Motion** - Smooth and beautiful animations that bring the UI to life.
*   **Lucide React** - Concise and clear icons.

> "Simplicity is the ultimate sophistication." - Bruce Lee. This was the guiding principle for my design.
