# ТЗ: нужен ИИ-агент «контент-завод», который автоматически делает и публикует короткие видео (20–60 сек) в Instagram Reels и YouTube Shorts.

**Функции: ИИ генерирует темы и сценарии, пишет текст диктора, создаёт видео (AI-аватар или на основе видео человека), подбирает фон и визуал, делает озвучку, автоматически монтирует ролик (9:16, субтитры, музыка).**

```
Также система генерирует описание, хештеги и автоматически публикует видео в соцсети по расписанию.
```

> Процесс: тема → сценарий → текст → генерация видео → монтаж → описание → автопостинг.

1. Цель — сделать «контент-фабрику», которая может выпускать 5–30 видео в день автоматически.
